## 📖 Description
OnCampusMart is a secure, localized e-commerce platform built exclusively for college students. It enables users to trade academic resources, electronics, and daily essentials with their peers. By implementing OTP-based college email verification and campus-specific filtering, it ensures a safe and trustworthy environment for all transactions.

## 🚀 Live Demo

   https://academic-resource-sharing-platform.vercel.app/

## 💻 Tech Stack
- **Frontend:** React.js, Bootstrap
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT

## ✨ Features
- **Campus Filtering:** View and filter products specific to your college or university.
- **Real-Time Chat:** built-in messaging system to negotiate and communicate with buyers/sellers.
- **Product Management:** Add, edit, and manage your product listings with image uploads.
- **Responsive Design:** Mobile-friendly UI for easy access on any device.
